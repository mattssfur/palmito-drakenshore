Mandamientos del Reino

Que todos los viajeros sigan el código sagrado

  1. No hostigar de cualquier manera para que todos tengan una experiencia buena en el server.
  2. Las Raids están permitidas.
  3. No ser llorón. Si hubiera algún problema, se puede reportar por Discord, WhatsApp, Instagram o en la web. Si el inconveniente es con un administrador, contacta a otro.
  4. No están permitidos los programas externos (hacks).
  5. Solo se puede tener 1 mascota por especie (Dinosaurios, Dragones, etc).
  6. Las Telelanterns deben estar en privado. Las públicas son solo puestas por administradores.
  7. No se puede poner Telelanterns a menos de 500 bloques de otra base.
  8. No está permitido jugar en grupos. El progreso es individual, aunque pueden colaborar sin formar teams.
  9. Las bases subterráneas no deben usar bloques que se confundan con los naturales.
  10. No romper bloques de la base colocados por otro jugador.
  11. No está permitido usar TNT, Creepers o similares con intención destructiva. No es un servidor anárquico.
  12. El pueblo es sagrado: no se puede destruir ni alterar.
  13. El mod A Magic Way tendrá ciertos hechizos prohibidos.
  14. No se permiten granjas técnicas que causen lag. Se fomenta la exploración y el esfuerzo.
  15. Se permite el uso de aldeanos para obtener libros.
  16. No se pueden crear tiendas. El mod de Item Shop es solo para administradores.
  17. Para comunicarse, usar el canal de Discord.
  18. Las fotos y videos del server deben subirse al canal de Discord como recuerdo del reino.
